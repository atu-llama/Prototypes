Evidence Evaluation Agent
Evaluates uploaded evidence for completeness, validity, and gaps — triages remediation priority automatically.
Draft
Trigger
Event — when evidence is uploaded
Prompt
Evidence evaluation
Show prompt
When evidence is uploaded, evaluate it against the following criteria: 1. Collection — Is this evidence mapped to a specific control? If not, flag for human review. 2. Completeness — Does it cover the required timeframe and scope for the mapped control? 3. Validation — Is the evidence authentic and from a verifiable source? 4. Gap identification — Are there missing or weak pieces of evidence? 5. Triage — Classify any gaps as: • Critical: evidence missing or invalid for an active control • High: evidence present but incomplete • Low: minor gaps, control coverage intact For Critical and High items, notify the assigned control owner via Slack. For items that cannot be mapped to a control, pause and flag for human review.
